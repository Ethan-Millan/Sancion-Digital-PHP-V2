# Sancion-Digital-PHP-V2

### Sistema de Gestión de Incidencias y Sanciones (Refactorizado)
Aplicación web desarrollada para la gestión, seguimiento y resolución de faltas al reglamento interno escolar. Este sistema permite a personal autorizado registrar infracciones y asignar horas de servicio comunitario, manteniendo un control estricto del cumplimiento.

---

## 🚀 Propósito del Proyecto
Este sistema nace como una evolución técnica de un proyecto previo. El objetivo principal es migrar un código "Legacy" hacia una arquitectura limpia y profesional, demostrando dominio en la lógica de negocio y la trazabilidad de datos.

## ✨ Características Principales
* **Arquitectura de Roles:** Control de acceso y permisos diferenciados para Alumnos, Profesores y Personal de Seguridad.
* **Gestión de Infracciones:** Módulo para levantar reportes de faltas en tiempo real asociados al perfil del infractor.
* **Seguimiento de Sanciones:** Asignación y control de horas de servicio comunitario (limpieza, composta, etc.).
* **Sistema de Notificaciones:** Alertas integradas para mantener informados a los usuarios.
* **Módulo de Reportes:** Generación de resúmenes de incidencias para evaluación administrativa.

## 🛠️ Stack Tecnológico & Buenas Prácticas
* **Backend:** PHP 8.x (Nativo/Vanilla) - Implementado bajo el paradigma de **Programación Orientada a Objetos (POO)**.
* **Seguridad:** Uso de **PDO** para la prevención de Inyecciones SQL y manejo seguro de sesiones.
* **Arquitectura:** Estructura de carpetas organizada para separar la lógica de negocio de la vista (preparación para entorno MVC).
* **Base de Datos:** MySQL con diseño relacional normalizado.
* **Frontend:** HTML5, CSS3, JavaScript.
